Welcome to the Eye Pressure Test

This project focuses on the development of an automated system for measuring eye pressure, a key indicator in diagnosing and monitoring eye health, particularly in detecting conditions like glaucoma. Glaucoma, a leading cause of blindness, often develops without noticeable symptoms, making early detection essential for effective management. This system aims to simplify the process of eye pressure testing through a user-friendly interface, providing a non-invasive and efficient method for individuals to assess their eye health.

The system is built using Flask for the backend, allowing users to easily access the test through a web interface. It includes features such as user authentication, allowing users to log in securely, and interactive forms for starting the test. The system records the eye pressure test results and provides a summary of the findings, offering insights into the potential need for further medical consultation.

Key Highlights:

  • User-Centric Design: Accessible and easy to use, with simple navigation for all users.
  • Real-Time Test Interface: Offers real-time feedback to guide users through the test.
  • Security Features: Incorporates login functionality to safeguard personal data.
  • Responsive and Attractive UI: A modern design using Bootstrap with animations to enhance the user experience.
  • Educational Content: Includes information about eye health and the importance of eye pressure testing.
